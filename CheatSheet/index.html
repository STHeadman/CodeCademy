Tag Definition Required Attribute

<html>

<html>Your code goes here.</html>

 Used to define everything between its brackets that it
is HTML code. 		None

<head>

<head>Your metadata goes here.</head>

 Used to define the head area of a web page. Filled with
metadata (not specifically shown on the web page)		 None

<title>

<title>Webpage Name</title>

 Used to define the title of the webpage. Always located
in the Head tags. The title will be shown in the tab title.		 None

<body>

<body>Webpage content goes here.</body>

 Ussed to define the body of the website. (what is shown
on the page)		 None

<nav>

<nav>using an Ul you can create a navbar here.</nav>

 Used to create a navbar to navigate your website None
